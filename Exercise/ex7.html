Exercise 7

สร้างโครงสร้างบทความข่าวด้วย Semantic HTML

Description

สิ่งที่โจทย์ต้องการ: ใช้แท็ก <article> สำหรับเนื้อหาหลักของบทความ ภายใน <article> ต้องมี <header> สำหรับส่วนหัวของบทความ (ชื่อเรื่อง ผู้เขียน วันที่เผยแพร่) ชื่อเรื่องบทความใช้แท็ก <h1>ข้อมูลผู้เขียนและวันที่เผยแพร่ใช้แท็กที่เหมาะสม เช่น <address> หรือ <div> ภายใน <header> เนื้อหาบทความแบ่งเป็นหลายย่อหน้า (<p>) และใช้แท็กสำหรับจัดรูปแบบข้อความ เช่น <strong>, <em> มีส่วนย่อยของเนื้อหาที่ใช้ <section> ภายในบทความ มีรายการที่ไม่เรียงลำดับ (<ul>) สำหรับหัวข้อย่อย หรือรายการที่เรียงลำดับ (<ol>) สำหรับขั้นตอน มีรูปภาพที่เกี่ยวข้องกับบทความพร้อมคำบรรยายภาพ โดยใช้แท็ก <figure> และ <figcaption> มีส่วนด้านข้าง (<aside>) ที่มีเนื้อหาที่เกี่ยวข้องแต่ไม่เป็นส่วนหนึ่งของเนื้อหาหลัก (เช่น บทความที่เกี่ยวข้อง, ลิงก์ที่น่าสนใจ) มีส่วนท้ายบทความ (<footer>) ที่มีข้อมูลลิขสิทธิ์หรือลิงก์ไปยังหน้าอื่น ตรวจสอบให้แน่ใจว่าได้ใช้ HTML5 Semantic Tags อย่างเหมาะสมตลอดทั้งเอกสาร
