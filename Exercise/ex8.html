Exercise 8

สร้างโครงสร้างบทความบล็อกเชิงความหมาย (Semantic Blog Post Layout)

Description

สิ่งที่โจทย์ต้องการ: ใช้แท็ก <article> เป็นคอนเทนเนอร์หลักสำหรับบทความบล็อกทั้งหมด ภายใน <article> ให้มีส่วนหัวเรื่องของบทความด้วยแท็ก <header> ซึ่งประกอบด้วย: ชื่อบทความ ใช้แท็ก <h1> และข้อมูลผู้เขียน (ชื่อผู้เขียน) และวันที่เผยแพร่ โดยใช้แท็ก <p> ร่วมกับแท็ก <time> สำหรับวันที่ แทรกรูปภาพที่เกี่ยวข้องกับบทความหรือรูปภาพผู้เขียน โดยใช้แท็ก <img> พร้อม alt text ที่เหมาะสม และพิจารณาใช้แท็ก <figure> และ <figcaption> หากเป็นรูปภาพที่มีคำอธิบายประกอบ ส่วนเนื้อหาหลักของบทความ ควรแบ่งเป็นย่อหน้า โดยใช้แท็ก <p> และอาจมีหัวข้อย่อยภายในเนื้อหาโดยใช้แท็กหัวเรื่องที่เหมาะสม (เช่น <h2>, <h3>) หากมีรายการข้อมูล เช่น รายการเคล็ดลับ ให้ใช้แท็กรายการที่เหมาะสม (<ul> หรือ <ol>) โค้ดที่ส่งมาต้องเป็น HTML เท่านั้น ไม่มี CSS หรือ JavaScript ในไฟล์
