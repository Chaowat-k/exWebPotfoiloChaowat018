Exercise 8

สร้างโครงสร้างบทความบล็อกเชิงความหมาย (Semantic Blog Post Layout)

Description

สิ่งที่โจทย์ต้องการ: ใช้ article เป็นคอนเทนเนอร์หลักของบทความ มี header ภายใน article สำหรับชื่อบทความ ผู้เขียน และวันที่เผยแพร่ ชื่อบทความใช้ h1 ข้อมูลผู้เขียนและวันที่ใช้ p และ time มีรูปภาพพร้อม alt และอาจใช้ figure และ figcaption เนื้อหาแบ่งเป็นย่อหน้า และมีหัวข้อย่อย h2 หรือ h3 มีรายการ ul หรือ ol หากมีข้อมูลเป็นข้อ ไฟล์ต้องเป็น HTML ล้วน ไม่มี CSS หรือ JavaScript
